473.astar Flags

These are flags that appear in the benchmark sources, categorized as to whether they are portability candidates, mandatory, or forbidden.

Sections

Selecting one of the following will take you directly to that section:

  • Portability Flags
  • Mandatory Flags
  • Forbidden Flags

Portability Flags

  • -DSPEC_CPU_BIG_ENDIAN
  • -DSPEC_CPU_LITTLE_ENDIAN
  • -DSPEC_CPU_WINDOWS_ICL
    • -DSPEC_CPU_BIG_ENDIAN
    • [-/]DSPEC_CPU_BIG_ENDIAN(?:=\S*)?\b

    • This macro indicates that the target system is big-endian.

    • -DSPEC_CPU_LITTLE_ENDIAN
    • [-/]DSPEC_CPU_LITTLE_ENDIAN(?:=\S*)?\b

    • This macro indicates that the target system is little-endian.

    • -DSPEC_CPU_WINDOWS_ICL
    • [-/]DSPEC_CPU_WINDOWS_ICL(?:=\S*)?\b

    • This macro indicates that the benchmark is compiled with Intel Windows C++ compiler.

Mandatory Flags

  • -DSPEC_CPU
  • -DSPEC_CPU_WINDOWS
    • -DSPEC_CPU
    • [-/]DSPEC_CPU(?:=\S*)?\b

    • This macro is set by the SPEC framework. It is not allowed to undefine this macro.

    • -DSPEC_CPU_WINDOWS
    • [-/]DSPEC_CPU_WINDOWS(?:=\S*)?\b

    • This macro indicates that the benchmark is compiled for Windows system.
