456.hmmer Flags

These are flags that appear in the benchmark sources, categorized as to whether they are portability candidates, mandatory, or forbidden.

Portability Flags

  • -DNOSTR
  • -DSEEK_SET
  • -DSRE_STRICT_ANSI
    • -DNOSTR
    • [-/]DNOSTR(?:=\S*)?\b

    • sets : extern char *strstr(char *s, char *subs);

    • -DSEEK_SET
    • [-/]DSEEK_SET(?:=\S*)?\b

    • If not already defined by other includes, include unistd.h to get definition.

    • -DSRE_STRICT_ANSI
    • [-/]DSRE_STRICT_ANSI(?:=\S*)?\b

    • Set by SPEC_CPU to use ANSI code.

Compiler Flags

  • -D__sgi
    • -D__sgi
    • [-/]D__sgi(?:=\S*)?\b

    • Helps choose IRIX-friendly code for pthreads. Not used in SPEC CPU2006.
