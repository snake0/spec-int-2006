    • Portability changes for IRIX

    • -DSPEC_CPU_LINUX
    • [-/]DSPEC_CPU_LINUX(?:=\S*)?\b

    • Portability changes for Linux

    • -DSPEC_CPU_MACOSX
    • [-/]DSPEC_CPU_MACOSX(?:=\S*)?\b

    • Portability changes for MacOS X

    • -DSPEC_CPU_COMPLEX_I
    • [-/]DSPEC_CPU_COMPLEX_I(?:=\S*)?\b

    • This flag may be used for Windows systems where IMAGINARY should be defined as "_Complex_I" instead of "__I__".

    • -DSPEC_CPU_NEED_COMPLEX_I
    • [-/]DSPEC_CPU_NEED_COMPLEX_I(?:=\S*)?\b

    • Portability changes for systems which need _Complex_I to be defined as 1.0fi

    • -DSPEC_CPU_NO_COMPLEX_H
    • [-/]DSPEC_CPU_NO_COMPLEX_H(?:=\S*)?\b

    • Defines this for systems which do not have complex.h

    • -DSPEC_CPU_NO_FABSF
    • [-/]DSPEC_CPU_NO_FABSF(?:=\S*)?\b

    • Define this for systems which do not have the fabsf function

    • -DSPEC_CPU_SOLARIS
    • [-/]DSPEC_CPU_SOLARIS(?:=\S*)?\b

    • Enables portability changes for Solaris

    • -Dinline
    • [-/]Dinline(?:=\S*)?\b

    • Allows user to define the inline keyword according to his compilation environment

Mandatory Flags

  • -DSPEC_CPU_WINDOWS
    • -DSPEC_CPU

    • [-/]DSPEC_CPU_WINDOWS(?:=\S*)?\b
