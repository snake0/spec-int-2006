    • -DXERCES_NO_ASM
    • [-/]DXERCES_NO_ASM(?:=\S*)?\b

    • This macro indicates that platform-specific assembly code is not to be used. It is set in Spec/object.pm and may not be changed.

    • -DXML_USE_NATIVE_TRANSCODER
    • [-/]DXML_USE_NATIVE_TRANSCODER(?:=\S*)?\b

    • This macro selects the "native" transcoder. It is set in Spec/object.pm and may not be changed.

    • -DXML_USE_INMEM_MESSAGELOADER
    • [-/]DXML_USE_INMEM_MESSAGELOADER(?:=\S*)?\b

    • This macro selects the in-memory message loader. It is set in Spec/object.pm and may not be changed.

    • -DXML_USE_PTHREADS
    • [-/]DXML_USE_PTHREADS(?:=\S*)?\b

    • This flag is set in Spec/object.pm, but has no effect on the compilation.

    • -DAPP_NO_THREADS
    • [-/]DAPP_NO_THREADS(?:=\S*)?\b

    • For every platform, this separates code that is the multi-threaded version fro mthe single threaded version. This is always defined in the object.pm to force the single threaded version.
