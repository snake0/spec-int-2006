436.cactusADM Flags

These are flags that appear in the benchmark sources, categorized as to whether they are portability candidates, mandatory, or forbidden.

Sections

Selecting one of the following will take you directly to that section:

  • Mandatory Flags
  • Forbidden Flags

Portability Flags

  • -DSPEC_CPU_LP64
  • -DSPEC_CPU_P64
    • -DSPEC_CPU_LP64
    • [-/]DSPEC_CPU_LP64(?:=\S*)?\b

    • This flag is used to indicate that the target system uses the LP64 data model, where longs and pointers are 64 bits wide.

    • -DSPEC_CPU_P64
    • [-/]DSPEC_CPU_P64(?:=\S*)?\b

    • This flag is used to indicate that the target system uses the P64 data model, where pointers are 64 bits wide, and longs are 32 bits wide.

Mandatory Flags

  • -Iinclude, -I../include
  • -DCCODE
    • -Iinclude, -I../include
    • -I(?:\.\./)?include\b

    • This flag is included automatically by Spec/object.pm.

    • -DCCODE
    • [-/]DCCODE(?:=\S*)?\b

    • This flag is set automatically in Spec/object.pm, and selects building of C code.
