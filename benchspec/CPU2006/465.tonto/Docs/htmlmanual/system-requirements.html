System requirements for TONTO

Apart from a Fortran compiler, you will need the following free software:

  • Netscape or another web browser

  • The tar program for extracting the distribution

  • The gunzip or bunzip2 program for uncompressing files

  • The gmake program for controlling the compilation

  • The perl program for the foo preprocessor

  • The cvs program and associated commands (only for developers)

  • The BLAS and LAPACK numerical libraries for linear algebra (only if not available on your system)

The above software must be installed and working before you start compilation.

If you are not doing development work and have a TONTO executable, then you do not need any of the above programs.
