433.milc Flags

These are flags that appear in the benchmark sources, categorized as to whether they are portability candidates, mandatory, or forbidden.

Sections

Selecting one of the following will take you directly to that section:

Portability Flags

  • -DSPEC_CPU_NEED_COMPLEX_TYPES
    • -DSPEC_CPU_NEED_COMPLEX_TYPES
    • [-/]DSPEC_CPU_NEED_COMPLEX_TYPES(?:=\S*)?\b

    • Complex inrinsic functions.

Mandatory Flags

  • -I.
  • -DCONGRAD_TMP_VECTORS
  • -DDSLASH_TMP_LINKS
  • -DFAST
  • -DFN
  • -DSPEC_CPU
  • -DSPEC_CPU_WINDOWS
    • -I.
    • -I\.(?:\s+|$)

    • This flag is included automatically by Spec/object.pm.

    • -DCONGRAD_TMP_VECTORS
    • [-/]DCONGRAD_TMP_VECTORS(?:=\S*)?\b

    • Optimizes performance for conjugate gradient by copying some vectors from site structure to temporary vectors.

    • -DDSLASH_TMP_LINKS
    • [-/]DDSLASH_TMP_LINKS(?:=\S*)?\b

    • Optimizes performance by copying gauge links from site structure to temporary arrays.

    • -DFAST
    • [-/]DFAST(?:=\S*)?\b

    • Performance enhancing code transformation.

    • -DFN
    • [-/]DFN(?:=\S*)?\b

    • Action contains both fat link in Naik (long link) terms.

    • -DSPEC_CPU
    • [-/]DSPEC_CPU(?:=\S*)?\b

    • Mandatory flag that sets a number of default portability flags.

    • -DSPEC_CPU_WINDOWS
    • [-/]DSPEC_CPU_WINDOWS(?:=\S*)?\b

    • Portability flag for windows.
